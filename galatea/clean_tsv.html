galatea.clean_tsv¶

clean_tsv(source, dest, row_diff_report_generator=None)¶

Clean tsv file high level function.

Parameters:

  • source (pathlib.Path) – source tsv file

  • dest (pathlib.Path) – output file name

  • row_diff_report_generator (TypeAliasForwardRef('galatea.clean_tsv.RowDiffReportGeneratorCallback') | None) –

    function for generating reports. When not provided, no report is generated.

    See RowDiffReportGeneratorCallback for details.

Return type:

None

RowDiffReportGeneratorCallback¶
