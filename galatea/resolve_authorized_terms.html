galatea.resolve_authorized_terms¶

resolve_authorized_terms(input_tsv, transformation_file, output_file, input_tsv_dialect=None, resolve_strategy=<function iter_resolved_terms>)¶

Resolve unauthorized terms to authorized terms in found tsv file.

Parameters:

  • input_tsv_dialect (Type[csv.Dialect] | csv.Dialect | None) – The dialect of the input tsv file. If None, it will attempt to guess

  • input_tsv (pathlib.Path) – The input tsv file to be transformed.

  • transformation_file (pathlib.Path) – The file to define transformations.

  • output_file (pathlib.Path) – Output file name.

  • resolve_strategy (galatea.resolve_authorized_terms.ResolveStrategyCallback) – Callable that returns a tuple of the original and new row.

Return type:

None
